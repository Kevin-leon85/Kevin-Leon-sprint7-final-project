# Kevin-Leon-sprint7-final-project. 
**Objetivo:** En este proyecto actuo como Data Analyst para ConnectaTel, una empresa de telecomunicaciones con presencia en México y Colombia. El objetivo principal es transformar datos crudos de tres fuentes distintas en insights accionables para optimizar la oferta comercial y mejorar la experiencia del usuario. Se exploran patrones de uso, demografía y otros.
**Dataset Utilizados:**                                                                                                       **a. plans.csv:** Contiene la configuración de los planes, incluyendo precios, minutos/mensajes incluidos y costos por servicios excedentes.                                                                                           **b.users_latam.csv:** Información demográfica de 4,000 usuarios, incluyendo edad, ciudad, fecha de registro y plan contratado.                                                                                                          **usage.csv:** Registro detallado de la actividad real, con la duración de cada llamada y el envío de mensajes.
**Etapas del analisis realizado:**                                                                               **a.Exploracion inicial:** Carga de datos y revision de estructuras para entender el contenido de cada dataset.            **b. Calidad de datos:** Identificacion y conteo de valores nulos, deteccion de sentinels y validacion de fechas fuera de rango.                                                                                                                      **c. Limpieza y estandarizacion:** Correccion de valores inconsistentes y conversion de tipos de datos para asegurar la consistencia estadistica.                                                                                                   **d. Estadistica descriptiva:** Calculo de medidas clave (media, mediana, percentiles) para identificar comportamientos tipicos y extremos.                                                                                                       **e. Analisis de outliers:** Uso de histogramas y boxplots junto con el metodo del IQR para detectar usuarios con consumos inusuales.                                                                                                               **f. Segmentacion:** Creacion de nuevas variables para clasificar a los usuarios por nivel de uso (Bajo, Medio, Alto) y grupo de edad (Joven, Adulto, Adulto Mayor).                                                                             **g. Insights ejecutivos:** Redaccion de conclusiones basadas en el analisis de datos.
